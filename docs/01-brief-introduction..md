 ## Introduction to Network Security 
 What is network security?
 Network security focuses specifically on protecting the infrastructure that causes communication and data exchange between devices.
 Understanding network vulnerabilities is crucial for protecting and keeping the infrastructure of an organization safe. 
 network security threats are as follows: 
 - malware
 - ddos attacks
 - phising and social engineering 
 - sql injection
 - zero day attacks
 - insider threats
 - man in the middle
