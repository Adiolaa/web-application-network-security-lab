# Web-application-network-security-lab

This is a coursework project done to introduce the basics of web application security and also the basics of network security  and how to mitigate them.

## Objective
The goals of this project are as follows:
- understand the basics of network security by learning the different types of network threats and ways to implement basic security measures.
- learn about common web application vulnerabilites by analysing a simple web application.

## Tools 
- WebGoat
- OWASP ZAP
- Wireshark
- Nmap
- pfSense
- ping(mobile).
